# Rental Buddy — 產品需求文件 (PRD)

> 版本：v1.0 | 目標平台：行動優先 PWA (Progressive Web App)
> 目標客群：20-35 歲女性，租屋新手

---

## 一、產品概述

**產品名稱**：Rental Buddy — 看房不踩雷

**核心問題**：
- 一般人租屋經驗不足，看房現場容易緊張遺漏重點
- 事後才發現問題（壁癌、電費超高、隔音差）已無法反悔
- 市面上沒有針對台灣租屋場景的好用 checklist 工具

**解決方案**：
一個可以在看房現場手機使用的 checklist 工具，分類清晰，每項都有說明，支援標記問題，最後輸出評估報告。

## 三、功能需求

### MVP（第一版）

#### 3.1 Checklist 核心
- [ ] 共 **5 大分類**，約 30 項目
  - 合約條件（租屋補助、水電費、押金等）
  - 設備狀態（冷氣、熱水、WiFi 等）
  - 安全採光（對外窗、壁癌、隔音等）
  - 生活機能（超市、通勤、垃圾等）
  - 環境鄰居（鄰居組成、房東風格等）
- [ ] 每項可展開看說明（避免用戶看不懂）
- [ ] 三種狀態：待確認 / 已確認 / 標記問題（⚑）
- [ ] 狀態儲存在 localStorage

#### 3.2 進度追蹤
- [ ] 頂部進度條（已確認 / 總項目數）
- [ ] 底部三欄統計（已確認 / 標記問題 / 待確認）
- [ ] 分類 Tab 顯示各類完成度

#### 3.3 看房記錄
- [ ] 可輸入看房地址（選填）
- [ ] 底部「查看報告」按鈕
- [ ] 報告顯示：完成度環形圖、問題清單、建議文字

#### 3.4 重置功能
- [ ] 每次看新房子前可重置所有記錄

## 四、設計規範

### 視覺風格
- **色調**：溫柔玫瑰粉 × 奶油白 × 鼠尾草綠（問題標記用）
- **字型**：Noto Serif TC（標題） × Noto Sans TC（內文）
- **圓角**：大量圓角，柔和感為主
- **動畫**：checklist 勾選有微動畫，報告modal 從底部滑入

### 設計原則
1. **手機優先**：所有互動考慮單手操作
2. **資訊密度適中**：看房現場注意力分散，UI 要清楚
3. **有溫度**：不要冰冷的 SaaS 感，像閨蜜推薦的工具
4. **快速上手**：第一次開就知道怎麼用

## 六、Checklist 資料

### 合約條件（7 項）
| 項目 | 說明提示 |
|------|---------|
| 租屋補助資格 | 確認房東是否配合申請青年租屋補貼（最高每月 2,500 元） |
| 水電費計算方式 | 確認是台電自訂價（每度 5-8 元）還是市價 |
| 押金金額 | 法定不得超過兩個月租金 |
| 管理費另計 | 每月可能額外 1,000-3,000 元 |
| 合約期限與違約金 | 確認最短租期與提前解約罰款 |
| 網路費用 | 是否含網路，能否自裝 |
| 可否養寵物 / 開伙 | 提前確認避免糾紛 |

### 設備狀態（8 項）
| 項目 | 說明提示 |
|------|---------|
| 冷氣是否為變頻款 | 舊型定頻超耗電，確認品牌年份 |
| 熱水能持續多久 | 打開等 30 秒測試穩定度 |
| 飲水機 / 濾水設備 | 確認有無，否則需自購 |
| WiFi 是否穩定 | 現場測速 |
| 洗曬衣空間 | 獨立或共用，費用如何計算 |
| 廚房設備 | 冰箱、微波爐、瓦斯爐狀態 |
| 工作空間 | 桌子大小、插座數量 |
| 停車空間 | 位置、費用、方便性 |

### 安全採光（6 項）
| 項目 | 說明提示 |
|------|---------|
| 對外窗方向 | 避免對走廊、暗巷，注意採光通風 |
| 是否有壁癌 / 漏水 | 檢查牆角、窗框附近 |
| 頂加 / 違建確認 | 夏天熱、噪音大，且有被拆風險 |
| 門鎖安全性 | 確認可否換鎖 |
| 滅火器 / 逃生設備 | 逃生路線是否暢通 |
| 隔音狀況 | 詢問鄰居作息或實際測試 |

### 生活機能（5 項）
| 項目 | 說明提示 |
|------|---------|
| 附近超市 / 便利商店 | 步行 5 分鐘內是否有採買地點 |
| 通勤路線確認 | 實際從房子走到捷運站計時 |
| 垃圾處理方式 | 垃圾車幾點來、廚餘如何處理 |
| 洗衣機排水問題 | 地排是否順暢 |
| 手機訊號強度 | 現場測試各大電信訊號 |

### 環境鄰居（4 項）
| 項目 | 說明提示 |
|------|---------|
| 鄰居組成 | 了解作息差異，評估影響 |
| 房東溝通風格 | 透過看房觀察是否好溝通 |
| 周邊噪音來源 | 確認夜市、酒吧、工地等 |
| 大樓管理員 | 安全性與服務評估 |
