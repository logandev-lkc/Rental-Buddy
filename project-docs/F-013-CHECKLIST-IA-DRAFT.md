## 3. 「三點擊」核心任務走查（現版粗估）

以下為 **自進入查核表頁** 起算的主要互動（點開選單並選一筆紀錄算 2～3 步）。

| 任務 | 現版步數估計 | 說明 |
|------|----------------|------|
| 切換紀錄 | 2～3 | 點紀錄 ▾ → 選另一筆（符合） |
| 改紀錄名稱 | 3～4 | 開選單 → 輸入 → 儲存（略超時需檢視是否合併步驟） |
| 編輯地址／月租 | 1 | 焦點進欄位（符合） |
| 開地圖選址 | 2 | 點「在地圖上選地址」→ 點地圖（符合） |
| 切分類並勾第一項 | 2～3 | 點分類 chip → 點項目（符合）；若預設「全部」需多一次點分類 |
| 前往報告 | 1 | 底部「查看報告」（符合） |

**待優化（連結 D2/D3）：** 若使用者目的是「我現在完成到哪」，視線需在 header／chip／底欄之間跳動；**資訊架構合併後**應再跑一次同表。
