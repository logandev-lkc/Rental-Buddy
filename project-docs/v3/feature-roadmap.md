## 建議切片路線圖

### Phase v3-A：現場一眼（摘要 + 條件列）

- **F-015**、**F-016**（可同一 sprint，若 IA 採「先排版後收斂 chips」則 F-015 先行）。

**預期成果**：手機寬度下摘要區主要資訊不需橫向捲動即可閱讀；戶型輸入路徑單一、備註可後填。

### Phase v3-B：分類導覽

- **F-017**

**預期成果**：無獨立「全部」雞肋感；多選分類行為與空狀態有規則；第六槽方向已定案（決策 **D-002** 候選）。

### Phase v3-C：狀態系統一致化

- **F-018**、**F-019**（規格／矩陣可先於實作定稿）

**預期成果**：抽樣題組符合公開聚合規則；互動稽核表與程式對齊（見 spec §5.2）。
