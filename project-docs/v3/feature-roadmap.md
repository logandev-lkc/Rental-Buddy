## 非本版範圍（Out of scope）

- 登入／雲端同步（延續 **D-001**；若改變須新決策條目與日期）。
- 無預算情境下承諾「後端同步」或帳號體系。
- 第三版**不以**大幅重做報告 PDF 版型為前提（若摘要欄位結構變更需對齊 v2／報告頁時另開條目）。
