## 五、與 App 實作對齊說明

App 已以本文件為準導入 **46** 題（`rb_*` id）、五軸中文名稱與 PDF 細節選項；篩選「高優先／一般／可後補」對應 `riskTier` 為 `must`／`should`／`later`。細節選項觸發評等同步時，使用 `riskOptionKeywords` 與 `nonRiskDetailOptions` 白名單（避免「無異味」類子字串誤判）。若 PDF 後續修訂，請同步更新 `app.ts` 並校對本檔表格。
