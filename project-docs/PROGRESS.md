# Rental Buddy 進度總表

最後更新：2026-05-09

## 目前產品結構

- 頂層頁面：`查核表`、`報告`
- 報告內檢視模式：
  - `手機版檢視`（友善閱讀）
  - `精簡版`（正式文件預覽）
- 匯出規則：`匯出文件（PDF）` 只輸出精簡版內容

## 已完成項目

### 1) 地圖與地址
- 查核表地址欄可開啟內嵌地圖，直接點選房源位置。
- 第一版使用 OpenStreetMap + Leaflet，不需要 Google API key。
- 點選地圖後會儲存 `latitude` / `longitude`，並用反查服務回填相似地址。
- 反查地址可再手動修正，避免免費服務地址格式不精準。

### 2) 核心資料與狀態
- 看房紀錄支援多筆，使用 `localStorage` 持久化。
- 每筆紀錄支援地址、月租金、各項 checklist 狀態與備註。
- 評分邏輯統一為 100 分制，並提供 A/B/C 評等。
- 評等門檻已調整為較嚴格版本：
  - A：85 以上
  - B：65-84
  - C：64 以下

### 3) 查核表體驗
- 新增月租金輸入欄位（與地址並列於基本資料區）。
- 紀錄切換、新增、刪除流程可用。
- 查核項可勾選/標記問題/展開補備註。
- 查核項支援快速狀態 chips 與高頻題目的細節選項 chips。
- 細節選項可自動同步 `checked` / `flagged` / `quickStatus`，降低現場操作步驟。

### 4) 報告頁體驗
- 改為「房屋評價」導向，不再使用性格測驗語氣。
- 報告頁可切換手機版檢視與精簡版預覽。
- 底欄改為兩大主按鈕（回到查核表、匯出 PDF）+ 可展開的「工具與 AI」區，減少一排過多按鈕。
- 手機版維持較溫馨、可快速掃描的資訊架構。

### 5) 精簡版與匯出
- 精簡版採檢驗單風格（分欄、表格、雷達、重點備註）。
- 匯出流程改為列印文件（非截圖），並鎖定輸出精簡版內容。
- 精簡版已加入 InBody 風格核心區塊：Rental Score、Confidence、分類分數、AI 分析總結、重要 Checklist 明細與下一步建議。
- 精簡版報表視覺再強化：表格與重要清單加入隔行底色、標題與欄位字距層級、下一步卡片圓角與留白優化，提升掃讀密度與穩定性。
- 精簡版在手機寬度的跑版問題已修正：
  - 單欄化
  - 雷達縮放
  - 表格可讀性調整
  - 操作按鈕避免擠壓

### 6) AI 報告整合
- 報告頁可複製完整 `reportData JSON`。
- 報告頁可複製一般 ChatGPT 報告生成 prompt。
- 報告頁可複製「可匯入 Prompt」，要求 ChatGPT 回傳固定 JSON。
- 可將 ChatGPT 回傳的 AI 分析 JSON 貼回前端，套用到 `AI 分析總結` 與 `下一步建議`。
- 若 AI JSON 缺漏、格式錯誤或被清除，報告仍會回到規則產生的 fallback。

### 7) 桌機版版面穩定性
- 調整桌機 header/footer/content 最大寬度與對齊。
- 改善捲動時覆蓋與視覺錯位問題。

### 8) PWA 與部署
- 已加入 PWA（manifest、icons、service worker）。
- 已修正 GitHub Pages 空白頁問題（`baseHref` 使用相對路徑）。
- 已同步 `docs/`，Pages 可部署最新版本。
- **F-010**：首次進入主流程後，符合條件時顯示「加到主畫面」提示；支援 Chromium 安裝對話框；iOS Safari 顯示「分享 → 加入主畫面」說明；提供「稍後」（約 7 天）與「不再顯示」（本機）。
- **F-011**：加入離線狀態提示列；離線顯示「資料仍保存在本機」與重新整理按鈕；恢復連線提示 4 秒後自動收合。
- service worker 快取策略微調：新增 OSM 圖磚與 Nominatim 反查快取群組，並加入「有新版本可用」提示與立即更新按鈕。
